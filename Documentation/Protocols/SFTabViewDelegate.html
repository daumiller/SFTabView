tabView:shouldSelectTab:

Sent to the delegate to allow or prohibit the specified tab to be selected.

- (BOOL)tabView:(SFTabView *)tabView shouldSelectTab:(CALayer *)tab

Parameters

tabView

The Tab View that sent the message.

tab

A tab contained in the Tab View.

When a tab is clicked by the user, this method will be called on the delegate. Returning NO will disallow that tab from being selected. Returning YES allows it to be selected.

Return Value

YES if the tab selection should be allowed, otherwise NO.

Discussion

Sent to the delegate to allow or prohibit the specified tab to be selected.
