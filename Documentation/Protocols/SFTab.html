SFTab Protocol Reference

Declared in SFTabView.h

Overview

The SFTab protocol defines the required methods that the CALayer subclass should implement for rendering and behaving as a tab.

desiredWidth

Returns desired width of tab.

- (CGFloat)desiredWidth

Discussion

Returns desired width of tab.

The defaultimplementation returns the width needed to display the tab with a non-truncated title.

setRepresentedObject:andWidth:

Tabs are are created or modified using the representedObject as model.

- (void)setRepresentedObject:(id)representedObject andWidth:(CGFloat)width

Parameters

representedObject

The Object passed to the layer that contain the information for drawing it (title, size, etc…).

width

The initial width of the tab.

The implementation of this method should read the information of the representedObject and build or update the CALayer according to it.

Discussion

Tabs are are created or modified using the representedObject as model.
